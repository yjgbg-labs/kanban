# Kanban

A full-stack Kanban application.

- `server/` - Rust web server (axum + sqlx + SQLite)
- `cli/` - Node.js CLI
- `web-ui/` - Remix.js web UI

## Getting started

### Server
cd server && cargo run

### CLI
cd cli && npm install && node index.js --help

### Web UI
cd web-ui && npm install && npm run dev
